## Feature

* Supports Python Virtual Environment
* Supports Git Branch
* Enabled beautified auto-completion feature

## Setup

To install:

```shell
mkdir -p ~/github/
cd ~/github
git clone git@github.com:littleq0903/littleq-zshrc.git
cd
ln -s ~/github/littleq-zshrc/.zshrc .
```
